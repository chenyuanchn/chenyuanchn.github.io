项目创建

  1. 使用idea开发，下载地址
  2. 打开idea，点击New Project
  3. 选择Spring Initializr,点击Next
  4. 填写Group、Artifact,点击Next
  5. 选择Web,勾选Web,点击Next
  6. 选择项目路径,点击Finish
